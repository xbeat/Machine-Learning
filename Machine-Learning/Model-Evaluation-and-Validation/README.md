# 📊 ML — Valutazione & Validazione dei Modelli

Metriche di valutazione, confusion matrix, ROC/AUC, cross-validation e tecniche per stimare l'affidabilità del modello.

## 📌 Argomenti chiave

- Confusion matrix: precision, recall, F1-score
- ROC curve e AUC per classificatori
- K-Fold, stratified e leave-one-out cross-validation
- Metriche per regressione: MAE, MSE, R²
- Silhouette score per il clustering
- Conformal prediction e calibrazione del modello
