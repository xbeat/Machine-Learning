# 🔧 ML — Feature Engineering & Preprocessing

Tecniche di feature engineering, scaling, encoding, selezione delle feature, gestione degli outlier e dei dataset sbilanciati.

## 📌 Argomenti chiave

- Feature scaling: StandardScaler, MinMax, Robust
- Encoding categorico: one-hot, target, hashing
- Feature selection: information gain, L1, probe method
- Gestione outlier e dati mancanti
- Dataset sbilanciati: SMOTE, reweighting
- Data augmentation e preprocessing pipeline
