# 🗄️ LLM — Vector Database & Embedding

Database vettoriali, quantizzazione, tokenizzazione, embedding LLM e ottimizzazione dell'inferenza per LLM in produzione.

## 📌 Argomenti chiave

- Vector database: Pinecone, FAISS, Qdrant
- Quantizzazione (GGUF, GGML, INT4/INT8)
- Tokenizzazione e KV caching
- Embedding models e scelta del modello
- Ottimizzazione dell'inferenza
- LLM locali con Ollama
